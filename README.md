# React Avanzado

## Comandos útiles

### Crear proyecto usando Expo

> npx create-expo-app@latest appexpo

### Crear proyecto con React Native CLI

Instalar react-native-cli

> npm install -g react-native-cli

Crear el proyecto 

npx react-native init appnative

### Crear proyecto React.js (solo web) usando Create React App (CRA)

> npx create-react-app web-cra

### Crear proyecto usando Parcel

Crear la carpeta y ejecutar `npm init -y`

Instalar React y React-DOM

> npm i react react-dom

### Crear proyecto usando Vite

> npm create vite@latest web-vite -- --template react
